# مشروع الواجهات الأمامية المتكامل (5 صفحات متجاوبة)

هذا المشروع هو تطبيق عملي وشامل لبناء موقع ويب متكامل يتكون من 5 صفحات تفاعلية متجاوبة بالكامل مع مختلف الشاشات، تم إنشاؤه كجزء من متطلبات التقييم البرمجي.

## 🚀 رابط المعاينة الحية
يمكنكم معاينة المشروع بشكل حي ومباشر عبر الرابط التالي:
[https://zhraaali.github.io/my-web-project/]

## 🛠️ التقنيات المستخدمة
* **HTML5:** لهيكلة الصفحات بشكل دلالي (Semantic HTML).
* **CSS3:** لإضافة التنسيقات المخصصة، وتأثيرات الـ Hover، ودعم الوضع الداكن.
* **Bootstrap 5:** لبناء نظام شبكة (Grid System) مرن ومتجاوب مع الجوال والتابلت.
* **JavaScript (Vanilla JS):** لإضافة تفاعلات ديناميكية مخصصة لكل صفحة بدون مكتبات خارجية.

## 📂 هيكلية ملفات المشروع
```text
my-web-project/
├── index.html         # الصفحة الرئيسية (السيرة الذاتية التفاعلية + الوضع الداكن)
├── blog.html          # صفحة المدونة التقنية (عرض النوافذ والتنبيهات)
├── store.html         # صفحة المتجر الوهمي (تتضمن عداد السلة الديناميكي)
├── calculator.html    # صفحة الحاسبة (برمجة حاسبة كاملة بـ JS)
├── contact.html       # نموذج الاستشارة (التحقق من صحة الحقول ومنع الإرسال الفارغ)
├── style.css          # ملف التنسيقات المخصصة
└── script.js          # ملف التفاعلات البرمجية لجميع الصفحات
